## 一、项目概述

PythonMentor 是一个基于 VSCode 插件的 Python 个性化学习助手，采用苏格拉底式教学法。系统通过 RAG 检索知识库、LLM 驱动的对话引擎、实时认知状态评估，为学生提供引导式教学而非直接给答案。

**技术栈**: FastAPI + ChromaDB + DashScope Embedding + Anthropic SDK (Mimo LLM) + VSCode Extension (TypeScript)

**9 阶段流程图**: 初始化 → 触发 → 预筛 → 数据检索 → Prompt 拼装 → LLM 推理 → 会话结束 → 知识库维护 → VSCode 插件

## 六、技术债务

| 问题 | 严重程度 | 说明 |
|------|----------|------|
| 无 Git 版本控制 | 高 | 无 .git 目录、无 .gitignore、无提交历史 |
| 双 chroma_db 目录 | 中 | 根目录和 python-backend 下各一份，可能索引不同步 |
| 文档过时 | 中 | README 目录结构与实际不符，项目进度清单标记为未完成的功能实际已完成 |
| 测试用自定义框架 | 低 | 未用 pytest，限制 CI 集成 |
| chat.js 从磁盘读取 | 低 | fs.readFileSync 而非 webpack 打包 |
| 无会话文件清理机制 | 低 | 服务器崩溃时孤立会话文件无自动清理 |
